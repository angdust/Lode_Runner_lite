Курсовой проект по JavaFX.

Он представляет один уровень игры "Lode Runner" по урощенным правилам:
1. Мобы двигаются по заданным траекториям, а не следуют за игроком.
2. Мобы не имеют возможность подбирать монеты.
3. Упрощенная анимация:
    а. текстуры игрока и остальных мобов упрощены.
    б. отсутствует анимация разрушения блока.
4. Разрушенные блоки не восстанавливаются.
5. Введено ограничение на количество разрушенных блоков (всего три штуки).
6. Невозможность удаления некоторых блоков (из-за изменений "3" и "4", их удаление не имеет смысла).

Некоторые ключевые приемы использовались из:
1. https://github.com/ZackLYChen/SuperMarioFinalProject (отрисовка карты и основы движения героя).
2. https://netopyr.com/2012/03/09/creating-a-sprite-animation-with-javafx/ (спрайтовая анимация для героя и мобов).

Для ознакомления с правилами игры использовалась онлайн версия
